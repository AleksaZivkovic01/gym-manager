About GymManager

GymManager is an application for managing training, members and trainers in a gym.

It was created to simplify organization and improve communication between all users of the gym.

What can you do as a guest?

  • view package offers
  • view trainers
  • basic information about trainings
  • introduction to the system

What do you get as a registered member?

  • booking and canceling sessions
  • view active membership
  • rating trainers
  • personalized notifications
